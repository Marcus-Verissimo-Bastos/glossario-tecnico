Glossário

Objetivo da Atividade:

Desenvolver um site de glossário técnico contendo tags HTML e propriedades CSS, com fins de consulta e apoio aos estudos. O site será autoral, com design próprio, e deverá ser publicado online.

Tecnologias Utilizadas:

  • HTML5
  • CSS3

Tags HTML

<code>

Cria um bloco de código.

<code>Seu código aqui</code>

<script>

Insere JavaScript no HTML.

<script>Seu código JS aqui.</script>

<div>

Usada para agrupar e dividir elementos.

<div>Seu texto aqui</div>

<mark>

Destaca o texto (estilo marca-texto).

<mark>Seu texto</mark>

<ul>

Define lista não ordenada.

<ul>Sua lista aqui</ul>

<s>

Risca o texto, mas mantém visível.

<s>Seu texto aqui</s>

<li>

Item de uma lista

<li>Seu item</li>

<input>

Define um campo de entrada de dados em um formulário

<input type="text" placeholder="">

<button>

Cria um botão.

<a href="link">Texto do botão</a>

<iframe>

Anexa outra página HTML a atual.

<iframe scr="site" widht>Seu texto aqui

<a>

Cria um link.

<a href="link">Seu texto aqui</a>

Propriedades CSS

filter

Aplica efeitos gráficos.

Filter: blur(5px);

clip-path

Recorta um elemento em forma geométrica.

clip-path: circle(50% at center)

backdrop-filter

Aplica efeitos gráficos ao fundo de um elemento.

backdrop-filter: blur(10px);

text-underline-offset

Define o espaçamento entre o sublinhado e o texto.

text-underline-offset: 4px;

text-emphasis

Aplica um estilo de ênfase a caracteres específicos.

text-emphasis: dot green;

user-select

Controla se o text dentro de um elemento pode ser selecionado.

user-select: none;

opacity

Define a opacidade de um elemento

opacity: 0.5;

z-index

Controla a sobreposição de elementos

z-index: 1;

image-rendering

Controla como as imagens são renderizadas.

image-rendering: pixelated;

box-shadow

Adiciona sobreamento ao redor de um elemento.

box-shadow: 10px 10px rgba(0, 0, 0, 0.3);
